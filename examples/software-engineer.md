# Software Engineer Example

```text
Use $interview-prep-coach to prepare me for a senior backend engineer interview.
First analyze my resume against the job description, then create a 3-day plan,
then run a system design mock interview one question at a time.
```

Useful inputs:

- Resume or project summary.
- Job description.
- Target level.
- Interview format: coding, system design, behavioral, or hiring manager.
